# AuditPulse

AI-Powered Website Audit Platform with Linear-style design.

## Features

- **AI-Powered Audits** - Comprehensive website analysis using AI
- **Performance Metrics** - Core Web Vitals, load times, and optimization insights
- **SEO Analysis** - Keyword rankings, content gaps, and meta tag optimization
- **Security Scanning** - SSL checks, security headers, and vulnerability detection
- **Competitive Intelligence** - Market sentiment and competitor analysis
- **PDF Reports** - Export detailed audit reports

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (Auth, Database, Edge Functions)
- **Charts**: Recharts
- **Build**: Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
